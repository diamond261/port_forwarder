# MCBE Forwarder

A fastest MCBE server forwarder.

## Usage

- Run normally

```bash
./ip_forward
```

- Run with config file

```bash
./ip_forward -c /path/to/config.json
```

- Run as daemon (background)

```bash
./ip_forward -d
```

- Generate systemd service file

```bash
./ip_forward -g
```

## Supported Platforms

- **Operating System:** Linux
- **Minecraft Version:** All the version

Run **build.sh** ,to build it automatically.
